Neural Architecture Search (NAS) automates the design of artificial neural networks (ANN), a technique that has outperformed manually designed architectures on tasks such as image classification and language modeling.

Search Space

The search space defines the principles of designing the architecture. It can be categorized into:

  • Macro architecture (entire network layout)
  • Micro/Cell architecture (designing basic building blocks)
def perform_nas(search_space, search_strategy, performance_estimation):
    best_arch = None
    best_perf = -1
    while not stop_condition():
        arch = search_strategy(search_space)
        perf = performance_estimation(arch)
        if perf > best_perf:
            best_arch = arch
            best_perf = perf
    return best_arch
